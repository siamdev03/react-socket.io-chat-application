Features

- **Real-Time Collaboration:** Socket.io bebohar kore ekebare instant text sync.
- **Rich Text Editing:** Quill.js diye pro-level text formatting (Bold, Italic, Lists, etc.).
- **Team Chat:** Sidebar-e real-time group chat persistence (refresh dileo message harabe na).
- **Presence System:** Kara kara online ache sheta dekha jay ebong typing indicator active thake.
- **Custom Avatars:** User-ra tader profile picture upload korte pare (Auto-compressed for performance).
- **Dark Mode:** Eyes-friendly dark ebong light mode toggle.
- **Data Persistence:** MongoDB Atlas-e document ebong chat history shob-e safe thake.
